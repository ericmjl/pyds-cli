## Requirements

- `pixi` on your PATH
- `git` on your PATH
- `uv` on your PATH

## Quickstart

Install from PyPI (package name **`pyds-cli`**; the CLI command is **`pyds`**):

```bash
pip install pyds-cli
```

Or install the latest revision from Git:

```bash
pip install git+https://github.com/ericmjl/pyds-cli
```
